# Image slots

These are the image slots on the site. Each corresponds to a `data-image-slot` placeholder in `index.html`. Drop files with these exact names in this folder and they'll be wired into the layout (currently the slots render as labeled placeholders until the images are added):

| Filename | Used in |
|---|---|
| `hero-runner.jpg` | Hero section background |
| `shoe-boston-13.jpg` | Shoe Data → Adidas Boston 13 panel |
| `shoe-cloudflyer.jpg` | Shoe Data → ON Cloudflyer panel |
| `gait-side-view.jpg` | Biomechanics → Video-Based Gait Notes |
| `gait-front-view.jpg` | Biomechanics → Video-Based Gait Notes |
| `gait-rear-view.jpg` | Biomechanics → Video-Based Gait Notes |

Any additional graphs/screenshots (e.g. Wahoo/treadmill data exports) can go here too — mention where you'd like them placed and they can be wired in next to the relevant metrics.
